Introdução ​
Primeiramente fiz esse sistema para a "Gincana: Fábrica do Noel", ele basicamente cria um clone do personagem no mapa que se move aleatóriamente.

Comandos de Plugin ​
Código: 
Clone <id do evento>
O evento que tem o id colocado no comando tornará-se o clone do personagem.

O script ficou muito booom!!!

O código ficou simples e legível.

Apenas um ponto que notei, foi o uso de chaves quando se tem apenas uma instrução dentro do "IF".

Código: 
    if (command === 'Clone') {
      createClone(parseInt(args[0]));
        }

Você poderia  omitir a chave quando se tem apenas uma instrução:

Código: 
    if (command === 'Clone')
      createClone(parseInt(args[0]));

Não vou retirar pontos por isso, pois não tem impacto na performance ou erro de codificação, é apenas uma melhoria na legibilidade do código.
