Introdução ​

Só postando aqui para questões de arquivamento.

Kazzter criou um tutorial de como alterar o formato do save , resolvi sugerir transformar aquilo num plugin pois desse jeito é menos evasivo. Eu tive algum tempo livre então fiz isso.

Comandos ​

Código: 
config text
Novo nome para o arquivo config.rpgsave.
Padrão: config

Código: 
global text
Novo nome para o arquivo global.rpgsave.
Padrão:t global

Código: 
file text
Novo nome para o arquivo file<número>.rpgsave.
Padrão: file
