Introdução

Hoje eu estava a brincar um pouco com requisições web e chamadas do sistema operacional em ruby e pensei: why not no rpg maker? E resolvi postar algumas das coisas que fiz aqui.
Para este post mostrarei como chamar um website usando o seu navegador padrão pelo seu jogo contanto que seja um dos RMs que usem ruby.

Comecemos

Vamos lá então, crie um "chamar script" ou chame ele por um script qualquer:
Código: 
system "start protocolo://url"

Uma breve explicação:
  • system é um método do ruby que executa um comando como se fosse o cmd.
  • start é um comando do windows que executa um comando, arquivo de lote ou um programa. Se ele ver que o argumento passado começa com um protocolo (como https://) então ele o abre no navegador padrão. Note que seria possível abrir arquivos locais no navegador usando "file:///" ou qualquer outro programa/comando.
  • Como podem ver no vídeo abaixo, sendo um comando do cmd ele abre o cmd (ou o powershell) antes de abrir o navegador. Isso pode ser evitado usando scripts em VB não mostrarei como fazer aqui já que pararia de ser uma "brincadeira descompromissada"
