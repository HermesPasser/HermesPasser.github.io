Introdução

Esse é um jogo que fiz para um evento por aí, eu não planejava publica-lo já que ele precisava do ambiente de desenvolvimento para funcionar, BEEEEEEM, precisava então why not? . Ele é um jogo relativamente rápido de ser terminado e bem linear.


História

Reconta o primeiro capítulo de Dracula, de Bram Stoker, com um twist no fim.


Download & Instruções

Download & Código-fonte no Github

1: Para jogar no windows: Baixe e descompacte o arquivo harker.zip e abra o seu terminal favorito (cmd/powershell/mingw) e execute o .exe. Ele não funcionará se você der duplo clique para abri-lo .

2: Jogar em sistemas *nix:
  • Baixe e descompacte o Source code (zip);
  • Instale o Python 3.7;
  • Instale PyYaml 3 usando o comando pip install pyyaml;
  • Rode o jogo usando python main.py , troque python por python3 se não funcionar.
