Introdução ​

O natal acabou e o Papai Noel voltou para seu trabalho de meio período, ajude-o a entregar as cervejas para aqueles que já esvaziaram o copo até o turno dele acabar!

Instruções ​

Sistema por eventos feito para o evento "É Natal 3".

Vá até o balção e pegue a bandeja com 5 cervejas, o que é o máximo que o Noel consegue carregar, após isso dê para cada uma das mesas uma e repita o processo até que o tempo se esgote.

Para mostrar o a voltagem em tempo real usei um plugin genérico chamado InfoDump que serve somente para mostrar um texto na tela. Abaixo deixarei uma explicação dos comandos usados no sistema/vídeo.

infodump enable - Mostra a janela;
infodump text texto para ser mostrado - Auto-explicativo. \n quebra a linha
