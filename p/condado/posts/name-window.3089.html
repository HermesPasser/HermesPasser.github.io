Introdução ​
Primeiramente fiz esse sistema para a "Gincana: Fábrica do Noel", ele basicamente mostra o nome do evento que está falando em uma janela. A janela some quando o texto é fechado.

Na primeira linha do "mostrar texto" deve conter #[name/id] sendo que name será o nome exibido e id o id do evento a janela ficará em cima. Coloque -1 no id para que seja em cima do personagem.

Comandos de plugin ​
Fazendo esse processo por comandos a janela não será fechada até que você ordene.

Código: 
NameWindow name eventId
Mostra a janela com o "name" para o evento que tenha o "eventId".

Código: 
NameWindow player name
Mostra a janela com o "name" para o jogador.

Código: 
NameWindow off
Desliga a janela.
