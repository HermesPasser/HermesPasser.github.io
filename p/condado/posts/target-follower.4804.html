Introdução ​
Este plugin faz que seu personagem use a função de movimento automático para ir até certo ponto do mapa após pisar num tile da aba tileset R.


Esse tileset aqui ​

A motivação desse plugin foi a dolorosa memória de meus tempos de XP quando eu tinha um evento de animação no meio do mapa no qual o personagem tinha que passar por determinado tile para acionar, mas para isso eu precisava fechar as passagem em volta para garantir que ele pisasse no tile certo estragando o design do mapa. Dentre as formas que usei para contornar esse problema na época foram replicar o evento nos células vizinhas (o que era dificultava a manutenção) e botar outros eventos que faziam meu personagem andar para o lugar certo (que eu precisava editar os passos para cada evento). Este plugin permite que você possa fazer mapas mais abertos e livres sem ter que se preocupar com o personagem chegar no lugar certo para ativar os eventos.


Nesse caso foi definido por um comando de plugin que quando o personagem pisar no tile com id 1 ele caminhará até o aquele evento. ​

Você poderá definir vários ids que o levarão para "alvos" diferentes e tais alvos serão descartados assim que o personagem chegar até eles. Vale notar que ele não caminhará até um lugar inválido e que se o caminha até chegar no alvo for muito complexo o personagem tentará um caminho inválido e ficará em loop. Não esqueça de descarta-los ao sair do mapa se planeja usar os mesmos ids em mapas diferentes.

Comandos ​

Código: 
settarget x y tile_id
Adiciona um alvo na celula XY ao pisar no tile com ID determinado. A posição deve ser válida e o alvo será descartado após ele pisar no id especificado e caminhar até o alvo.

Código: 
deletetarget tile_id
Remove o alvo com o ID selecionado sem ter pisado nele e caminhado até o alvo.

Código: 
cleartargets
Descarta todos os alvos definidos.
