Introdução ​

Sistema de cozinha para o evento de natal do condado. Ele permite que você defina produtos a serem cozinhados e receitas para serem usadas, além de uma maneira elegante de observar as receitas (cena livro de cozinha).

Instruções ​

Adicione um notação JSON como na mostrada abaixo no item que você quer que seja produzido usando o sistema:

Código: 
[  
   { "id": 01, "qnt": 1 , "txt": "%qnt colheres de %item"},
   { "id": 02, "qnt": 99, "name": "açucar" }
]

Note que a notação acima é a única coisa que deve ser adiciona no campo notas.

Cada array (conteúdo dentro dos {}) é um ingrediente, você pode colocar o quantos quiser.
"id" é referente ao id do item que você quer usar como ingrediente, note que o número não é posto em entre aspas por ser um número. Um id inválido exibirá um erro.

"qnt" é a quantidade necessária para se produzir o alimento.

"name" caso que você queira que outro nome que não seja o nome do banco de dados apareça no livro de cozinha. É opcional, se omitido ficará o nome do item.

"txt" é um texto customizado para aparecer no livro de receitas, %qnt será substituído pela quantidade necessária e %item pelo nome do item. É opcional, se omitido ficará "%qnt of %item".

Comandos de plugin :

cook book id mostra a receita no livro de receitas, sendo que "id" deve ser trocado pelo id do item com a receita. Se o item não tiver uma receita um erro será exibido.

cook id verifica se você tem os ingredientes em quantidades suficientes e se sim lhe retorna o produto debitando a quantidade de ingredientes usados de seu inventário. Nada ocorrerá que não houverem ingredientes o suficientes no inventário.
